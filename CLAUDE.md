# Claude Instructions

When working in this repository:

- Keep examples generic and public-safe.
- Do not add real IP addresses, tokens, private hostnames, private chat IDs, usernames, or production config.
- Prefer conservative permissions and explicit human approvals.
- If you add a new auto-repair action, document why it is safe and how it is verified.
- Treat Hermes provider credentials, messaging tokens, auth files, memories, sessions, and chat logs as sensitive.
